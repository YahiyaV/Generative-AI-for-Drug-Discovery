Generative AI for Drug Discovery

Design novel drug candidates with deep generative models and predict their molecular properties using Graph Neural Networks

Project Pipeline

This system uses a two-stage AI pipeline for drug molecule design:

  1. SMILES VAE — A Variational Autoencoder trained on character-level SMILES strings learns the chemical latent space. Sampling from this space generates novel, chemically diverse molecules.
  2. GNN Property Predictor — A Graph Convolutional Network takes molecular graphs (atoms = nodes, bonds = edges) and predicts key drug properties: molecular weight, LogP, TPSA, and QED.
